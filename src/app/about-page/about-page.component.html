About this Application

This application was written as a technical exercise to try out the Clarity Design Framework in Angular. I wanted to learn more about it, and build something relevant. So I decided to build a web app to give myself an easy to digest data about COVID-19. I felt most data visualizations didn't give a complete picture of the data.

Data is obtained from The COVID Tracking Project. Please refer to that project for any questions about the data or it's accuracy.
